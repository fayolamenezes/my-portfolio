Meal Planner Web App (React Version)

A simple yet effective meal planning application that helps users organize their weekly meals using selected recipes and maintain consistency in their diet.

Key Features:

  • Interactive day-wise meal planning interface
  • Dropdown-based recipe selection for each meal
  • Automatic local storage of weekly plans
  • Clean, responsive design for usability

Technologies Used:

React, HTML, CSS, JavaScript, LocalStorage, Vercel
