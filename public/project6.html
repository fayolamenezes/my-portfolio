Recipe Manager Web App

A feature-rich web application to manage recipes, plan meals, and generate grocery lists—designed for ease of use and practical daily cooking needs.

Key Features:

  • Recipe creation with image uploads and category filtering
  • Live search and edit/delete functionality for recipes
  • Weekly meal planner with dropdown selections
  • Grocery list generation based on planned meals

Technologies Used:

HTML, CSS, JavaScript, LocalStorage, Vercel
